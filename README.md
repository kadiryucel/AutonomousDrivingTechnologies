"# AutonomousDrivingTechnologies" 

The Autonomous Driving Technologies Specialization Program aims to train data processing and integration specialists by collaborating with leading companies in the sector to meet the need for qualified human resources in the field of artificial intelligence. The program is divided into two parts: basic training and specialization training. Basic training provides the fundamental skills necessary for specialization training, while specialization training combines theoretical knowledge with practical application.

In this context, the code for my assignments and final project from the training I received in 2025 is included.

Artificial intelligence was used to support optimization and refinement during the code writing process.
